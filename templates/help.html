Welcome To Indoor Navigation Help API


Here you can read documentation about various other API served and how to use them.

1. Accesing FireBase APIs


URL => '/mapALL' [GET]
Returns a JSON response with all available building names in it, keyd with 'result'.
Example Output => { 'result' : ['building1Name', 'building2Name', 'building3Name']}

URL => '/floorALL/<building>' [GET]
Params => Building name[GET/URL]
Returns a JSON response with all available floor names in passed building, keyd with 'result'.
Example Output => { 'result' : ['floor0', 'floor1']}

URL => '/roomALL/<building>/<floor>' [GET]
Params => Building name[GET/URL], Floor Name[GET/URL]
Returns a JSON response with all available room names in passed building and floor, keyd with 'result'.
Example Output => { 'result' : ['shop1', 'shop2']}

URL => '/room/<building>/<floor>/<room>' [GET]
Params => Building name[GET/URL], Floor Name[GET/URL], Room Name[GET/URL]
Returns a JSON response with room's px data in passed building and floor, keyd with 'result'
Example Output => { 'result' : [69, 420]}

2. Accesing Trilateration API


URL => '/locate/<building>' [POST]
Params => Building name[GET/URL], Router Data(Min 4 required)[POST/JSON]
Returns a JSON response with user's lat long(for now, have to change it to px) in passed building, keyd with 'result'.
(have to add feature to tell user youre not at that bulding)
Example Input => { 'router' : {
'MAC1' : -57,
'MAC2' : -50,
'MAC3' : -60,
'MAC4' : -55
}}
Example Output => { 'result' : ['building1Name', 'building2Name', 'building3Name']}

3. Accesing Image APIs


URL => '/route/<building>/<floor>' [GET]
Params => Building name[GET/URL], Floor Name[GET/URL]
Returns a JSON response with map image encoded in b64 for passed building and floor, keyd with 'result'.
Example Output => { 'result' : 'encodedImg'}

URL => '/locate/<building>/<floor>' [POST]
Params => Building name[GET/URL], Floor Name[GET/URL], Routing Locations[POST/JSON]
Returns a JSON response with route img encoded in b64 for passed params, keyd with 'result'.
Example Input => {
'origin' : [69, 69],
'destination' : [420, 420]
}
Example Output => { 'result' : ['building1Name', 'building2Name', 'building3Name']}
